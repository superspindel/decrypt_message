# decrypt_message

If null termination is removed, the program will panic at runtime since the recursive end point does not happen.

## To download

1. `git clone https://github.com/superspindel/decrypt_message.git`
2. `cd decrypt_message`
3. `git checkout 3a`

## To run

1. `cargo run`

### Optional

1. `cargo build`
2. `cargo run`
